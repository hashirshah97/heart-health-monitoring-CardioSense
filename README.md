## 📌 About
**CardioSense** is a heart disease prediction system that uses Machine Learning to analyze cardiovascular health data and predict disease risk.

---

## 📁 Project Structure
```
Heart Disease/
│
├── 📂 dataset/          # Heart disease dataset
├── 📂 Notebook/
│   └── Heart Disease Prediction.ipynb
├── app.py               # Main application file
├── .gitignore
└── README.md
```

## 🚀 How To Run

```bash
# Step 1: Clone the repo
git clone https://github.com/hashirshah97/heart-health-monitoring-CardioSense.git

# Step 2: Install libraries
pip install -r requirements.txt

# Step 3: Run the app
python app.py
```

Or open **Heart Disease Prediction.ipynb** in Jupyter Notebook and run all cells.
